# Grade Interativa — Eng. de Computação UFC

Fluxograma interativo para acompanhar o progresso ao longo do curso de Engenharia de Computação na UFC Fortaleza.

🔗 **Acesse aqui:** [ysmmfe.github.io/grade-eng-comp-ufc](https://ysmmfe.github.io/grade-eng-comp-ufc/)

## Como usar

- **Clique** em uma disciplina para marcá-la como concluída (fica verde)
- **Clique novamente** para desmarcar
- **Passe o mouse** sobre uma disciplina para ver seus pré-requisitos (vermelho) e o que ela desbloqueia (verde)
- Disciplinas com **borda azul** já estão liberadas (todos os pré-requisitos cumpridos)
- **Optativas** aparecem como cards tracejados nos semestres 7–10 — clique para escolher no catálogo
- **Extensão** e **Atividades Complementares** ficam na seção abaixo da grade
- Seu progresso é **salvo automaticamente** no navegador (localStorage)

## Dados

Todas as disciplinas, pré-requisitos e cargas horárias foram extraídos do **PPC 2022/2023** e do **SIGAA** do curso de Engenharia de Computação da UFC.
